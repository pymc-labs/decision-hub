# Missing Standard Community Documentation

## Description
The repository is missing standard open-source community documentation files:
- `CONTRIBUTING.md`: Currently, the README points to `CLAUDE.md`, which is tailored for AI agents, not human contributors.
- `CODE_OF_CONDUCT.md`: No code of conduct is present.
- `SECURITY.md`: No policy for reporting security vulnerabilities.

## Impact
- **Contributors**: Human contributors may feel unwelcome or unsure how to contribute.
- **Safety**: Lack of a Code of Conduct can deter participation.
- **Security**: Security researchers may not know how to responsibly disclose vulnerabilities.

## Recommendation
- Create a `CONTRIBUTING.md` based on `CLAUDE.md` but formatted for humans (e.g., how to run tests, submit PRs).
- Add a standard `CODE_OF_CONDUCT.md` (e.g., Contributor Covenant).
- Add a `SECURITY.md` with contact details (e.g., email) for security reports.
